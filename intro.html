3 Funciones

3.1 Todo en R es un objeto

class(sum)

3.2 Para definir una función usamos la función function

myFun <- function(x, y) x + y
myFun(3, 4)
class(myFun)

3.3 Definir una función a partir de funciones

foo  <-  function(x, ...){
  mx <- mean(x, ...)
  medx <- median(x, ...)
  sdx <- sd(x, ...)
  c(mx, medx, sdx)
  }

O en forma resumida: 

foo <- function(x, ...){c(mean(x, ...), median(x, ...), sd(x, ...))}

3.4 Y ahora usamos la función con vectores

foo(1:10)

rnorm(100)
help(rnorm)
foo(rnorm(1e5))

3.5 Y también funciona con matrices

rowMeans(M)
apply(M, 1, foo)
colMeans(M)
apply(M, 2, foo)
